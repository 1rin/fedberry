FedBerry is a Fedora Remix specifically built for use with the Raspberry PiÂ® 2 Model B computer.

As Fedora does not (yet) support the Raspberry Pi 2 Model B (RPi2), FedBerry was created so people could easily install and use Fedora on the RPi2.

FedBerry is still in its infancy with 3 active developers working on the project. If you wish to contribute, please feel free to join us.

The FedBerry team would like to say a big thank you to all the Fedora developers that make this remix possible. We appeciate your dedication and hard work and are grateful that you make it freely available to all.

  • Minimal and XFCE release

    Both these releases are close to 'stock' arm releases from Fedora. Where possible, they endeavour to stick to default Fedora release settings.

    The root partition is automatically re-sized when the OS is first booted. After re-sizing, it will automatically reboot. After rebooting the system will launch the 'initial-setup' utility. For graphical images this will occur on the display, for minimal images this will occur on the serial console. Failure to complete the initial-setup will prevent logging into the system.

  • Mini release

    This release is for ADVANCED users only, as it differs significantly from standard default Fedora releases. It is a small stripped 'bare bones' remix with some optimizations specifically tailored towards RPi2 usage. Its main goals are to have a small footprint, fast boot times and to reduce the number of writes to the SD card (within reason).

    So you are aware of its limitations, please ensure you read the release notes before using this release.

    The root partition is NOT automatically re-sized at boot and the initial-setup utility is not used for this release.
