FedBerry is Fedora Remix specially created for Raspberry PI 2 arm based computer.

FedBerry created by Vaughan (agrez) in October 2015 as home project for fun and practice.

In January 2016 Gor (Gormartsen) joined project.

FedBerry is a first RPM based linux distributive to support Raspberry PI 2 and it was a main reason to create it.

Right now FedBerry community are 2 people who are having fun with Raspberry PI 2. Feel free to join us.
